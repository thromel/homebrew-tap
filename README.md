# thromel/homebrew-tap

Homebrew tap for `ctxhelm`, a local, read-only context compiler for AI coding agents.

## Install

```bash
brew tap thromel/tap
brew install ctxhelm
ctxhelm --version
```

## Current Formula

- Formula: `Formula/ctxhelm.rb`
- Version: `1.1.9`
- Release: `https://github.com/thromel/ctxhelm/releases/tag/v1.1.9`
- Platform: Apple Silicon macOS archive

The formula installs the prebuilt public archive and runs `ctxhelm --version`
and `ctxhelm --help` in the Homebrew test block.
